# Blinkit-Sales-Analysis using Python

## Project Overview
This project analyzes Blinkit's sales performance, customer satisfaction, and inventory distribution using Python.

The objective is to identify key business insights and opportunities for optimization through data analysis and visualization.

## Business Requirements
The analysis focuses on the following KPIs:

1. Total Sales
2. Average Sales
3. Number of Items Sold
4. Average Rating

## KPIs Calculated

| KPI | Description |
|-------|------------|
| Total Sales | Overall revenue generated |
| Average Sales | Average revenue per transaction |
| Number of Items | Total count of items sold |
| Average Rating | Average customer rating |

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset Information
The dataset contains information regarding:
- Item Type
- Fat Content
- Outlet Size
- Outlet Location
- Outlet Establishment Year
- Sales
- Ratings

## Analysis Performed

### 1. Total Sales by Fat Content

**Objective:**
Analyze the impact of fat content on total sales.

**Visualization:**
Donut Chart

### 2. Total Sales by Item Type

**Objective:**
Identify top-performing item categories.

**Visualization:**
Bar Chart

### 3. Fat Content by Outlet for Total Sales

**Objective:**
Compare sales across outlets segmented by fat content.

**Visualization:**
Stacked Column Chart

### 4. Total Sales by Outlet Establishment

**Objective:**
Evaluate how outlet age influences sales.

**Visualization:**
Line Chart

### 5. Sales by Outlet Size

**Objective:**
Analyze the relationship between outlet size and sales.

**Visualization:**
Pie Chart

### 6. Sales by Outlet Location

**Objective:**
Assess geographic distribution of sales.

**Visualization:**
Funnel Chart

## Key Insights

### Sales Performance
- Low Fat products generated higher sales than Regular products.
- Fruits & Vegetables and Snack Foods were the highest-selling categories.

### Outlet Analysis
- Medium-sized outlets contributed the highest revenue.
- Tier 3 locations generated the maximum sales.

### Customer Satisfaction
- Average customer rating remained consistent across most product categories.

### Business Recommendations
- Increase inventory for high-demand categories.
- Expand operations in high-performing Tier 3 locations.
- Focus promotional campaigns on top-selling products.
